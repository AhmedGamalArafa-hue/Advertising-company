# IDEA Company - Advertising Agency Website 🚀
## (منصة وكالة إبداعية حديثة ومتكاملة)

تحويل موقع وكالة إعلانية من تصميم HTML تقليدي إلى تجربة ويب حديثة، احترافية، وتفاعلية (Modern Creative Agency Platform). يقدم الموقع معرض أعمال متكامل، نظام إدارة محتوى (CMS) مبسط، وتجربة مستخدم متميزة مبنية على أرقى معايير التصميم العالمي (Enterprise-Grade UI/UX).

---

## 🌟 المميزات المضافة حديثاً (Premium Upgrades)

### 📱 نظام الملاحة التفاعلي المتجاوب بالكامل (Mobile-Responsive Glass Navbar)
* **قائمة منسدلة زجاجية (Glassmorphic Dropdown):** تم توحيد شريط التنقل العلوي في جميع صفحات الموقع. عند فتحه على الهواتف الذكية، تنسدل قائمة خيارات زجاجية غاية في النعومة بانزلاق انسيابي (`navSlideDown`) وسرعة استجابة فائقة.
* **زر هامبرغر متحرك بالـ CSS (Animated Hamburger to X):** تصميم هندسي بالبكسل بدون استخدام نصوص، حيث تتحول الخطوط الثلاثة للهامبرغر ديناميكيًا إلى علامة `✕` متناسقة ومتحركة بسلاسة عند فتح القائمة وتعود لوضعها الأصلي عند الإغلاق.
* **توحيد روابط التنقل:** تحتوي جميع الصفحات الآن على وصول كامل وموحد لكافة أقسام الموقع (`Home` • `Logo` • `Business Cards` • `Request`).

### 💎 الواجهة الزجاجية ثلاثية الأبعاد لتسجيل الدخول (Glassmorphic Login Card)
* **تأثير الزجاج الحقيقي:** تم تصميم صندوق تسجيل الدخول بلوحة التحكم بتأثير زجاجي متقدم يعتمد على تشتيت الضوء (`backdrop-filter: blur(25px) saturate(180%)`) وإطار أبيض مشع يحاكي انعكاس الضوء على حواف الزجاج الحقيقي.
* **نبض الإضاءة الخلفية الشفافة (Backlight Glow):** تم وضع عنصر متوهج بلون أزرق متدرج (`.login-glow`) خلف البطاقة الزجاجية مباشرة ينبض ببطء (`pulseGlow`) ليعطي عمقاً بصرياً ثلاثي الأبعاد يبهر المستخدمين.

### 🌗 إصلاح وتحديث حقول الإدخال والوضع المضيء (Smart Inputs & Light Mode)
* **متغيرات ديناميكية (Theme Variables):** تم ربط ألوان الحقول بمتغيرات CSS (`--input-bg` و `--input-color`) لتتغير فوراً وبنعومة مع تبديل وضع الإضاءة.
* **رؤية ممتازة في وضع الضوء:** تم إلغاء اللون الداكن الثابت، لتصبح الحقول في الوضع المضيء ذات خلفية رمادية ناعمة ونصوص واضحة وداكنة تماماً، مما يحل مشاكل سهولة القراءة السابقة.

### 📂 أزرار وعناصر تفاعلية مخصصة (Custom Interactive Elements)
* **أزرار إطار زجاجي (Outline Buttons):** قمنا بإنشاء وتطبيق فئة أزرار الإطار الشفاف `.btn-outline` ذات التوهج والتأثير الحركي الممتلئ باللون الأزرق على أزرار "Save Project" و "Send Request" و "Login" و "Back to Website".
* **زر رفع الصور الفاخر (Premium File Input):** قمنا بإلغاء زر الرفع الافتراضي القبيح، وتصميمه بستايل منطقة رفع متقطعة الإطار (`dashed border`) مع زر مخصص باللون الأزرق يتوهج عند التفاعل.
* **توحيد حجم الأزرار في لوحة التحكم:** زر تسجيل الخروج `Logout` وزر العودة للموقع `Back to Website` أصبح لهما نفس الأبعاد والدائرية تماماً لتحقيق اتساق هندسي مريح للعين.

### 📐 ضبط هيكل معرض الأعمال والصفحات الفرعية (Optimized Showcases)
* **تصميم Logo Design:** تم ضبط حاوية Showcase وزيادة العرض البصري وتوسيط الصور الطولية لمنع استطالتها أو تشويهها تحت الـ Navbar الزجاجي الطافي.
* **تصميم Business Cards:** تحويل الهيكل إلى تخطيط أفقي مخصص يعرض الكروت بحجمها الطبيعي ونسبة أبعادها الواقعية، مع تنظيم قائمة الفوائد في شبكة ثنائية الأعمدة فائقة الأناقة.
* **إلغاء وحذف الأقسام غير المستخدمة:** تم حذف صفحات الكروت الشخصية ودعوات الزفاف (`bookmarks` و `wedding-invitations`) بالكامل من بنية الموقع وقواعد البيانات البرمجية والروابط والقوائم لتنظيف الكود وتسريعه.
* **تصفية الفوتر الموحد:** تم تطبيق فوتر احترافي موحد، متكامل، ومضغوط المسافات في جميع صفحات الموقع لربط أقسام الموقع بأناقة متناهية.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5:** لهيكلة المحتوى بشكل Semantic ومحسّن لمحركات البحث (SEO).
* **CSS3 (Vanilla):** للتنسيق المتقدم، تأثيرات الـ Glassmorphism، الحركات التفاعلية، ونظام الألوان المزدوج المتطور.
* **JavaScript (Vanilla):** لإدارة التفاعلات الحية، القوائم المنسدلة، تبديل الثيم، ونظام الـ CMS المحلي للوحة التحكم.
* **LocalStorage:** لحفظ بيانات الجلسة البرمجية، اختيار الثيم (Dark/Light)، والمشاريع المضافة حديثاً دون الحاجة لقاعدة بيانات خارجية.

## 📁 هيكلية مجلدات المشروع

* 📄 `index.html`: الصفحة الرئيسية للموقع وتضم معرض الأعمال وشبكة ثنائية لعرض الخدمات الأساسية.
* 📄 `logo-design.html`: صفحة عرض تفاصيل ومشاريع قسم تصميم الشعارات (معرض عمودي متناسق).
* 📄 `business-cards.html`: صفحة عرض تفاصيل ومشاريع قسم كروت الأعمال (تخطيط أفقي مع شبكة مميزات ثنائية).
* 📄 `application-form.html`: نموذج طلب الخدمة وتعبئة البيانات بشكل فخم وخلفيات ديناميكية متوافقة مع الثيم.
* 📄 `dashboard.html`: لوحة تحكم الإدارة (تضم إحصائيات حية، نموذج رفع الصور الفاخر، والتحكم التام بالمشاريع مع حماية الدخول).
* 📁 `Project Phtots/`: مجلد يحتوي على كافة الصور والأيقونات الفاخرة للخدمات والموقع.
* 📄 `style.css`: نظام التصميم الفني الشامل، ويضم متغيرات الألوان، حركات الأنيميشن التفاعلية، وتصميمات الهواتف.
* 📄 `script.js`: المنطق البرمجي الأساسي للتحريك، تبديل الثيم، القوائم التفاعلية والتحكم.
* 📄 `projects.js`: قاعدة بيانات المشاريع الافتراضية للوكالة.
* 📄 `README.md`: دليل توثيق المشروع والنسخ.
